# Rental App
## About
An app for renters to review the properties they have resided in.  
An app for landlords to review the renters that have resided in their properties.

## Development
### Python Virtual Environment
Activate venv
```bash
source .venv/bin/activate
```

Deactivate venv
```bash
deactivate
```

### Django
Start server
```bash
docker-compose up
```
